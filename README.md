# 🇲🇲 Myanmar Pyidaungsu Font Typing Software

A simple and efficient typing software designed for **Myanmar Unicode (Pyidaungsu)** input across multiple platforms. This tool helps users easily type, read, and work with Myanmar text on **Windows, macOS, and Linux**.

---

## ✨ Features

* 📝 Full support for **Myanmar Unicode (Pyidaungsu font)**
* ⚡ Fast and lightweight typing experience
* 💻 Cross-platform compatibility
* 🔤 Easy keyboard input setup
* 🎯 Clean and user-friendly interface
* 📦 Simple installation process

---

## 🚀 Installation

### Windows

```bash
1. Download the .exe file
2. Run the installer
3. Follow setup instructions
4. Start typing in Myanmar!
```

### macOS

```bash
1. Download the .dmg file
2. Open and drag to Applications
3. Enable input source in Keyboard Settings
```

### Linux

```bash
# Example for Debian/Ubuntu
sudo dpkg -i myanmar-typing.deb
```

---

## ⌨️ Usage

1. Enable the Myanmar keyboard/input method
2. Select **Pyidaungsu Unicode**
3. Start typing in any application
